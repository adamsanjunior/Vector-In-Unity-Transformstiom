# Vector3 Object Transformation in Unity

This project demonstrates the use of the Vector3 data type in Unity to perform
translation, rotation, and scaling of a 3D object using C# scripting.

## Features
- Continuous movement using Vector3
- Continuous rotation using Vector3
- Gradual scaling with a defined limit
- Frame-rate independent motion using Time.deltaTime

## Tools
- Unity 3D
- C#
- Unity Hub
